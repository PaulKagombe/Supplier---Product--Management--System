Supplier & Product Management System

A Java Object-Oriented Programming (OOP) application designed to manage suppliers, products (materials), customers, and sales operations in a construction warehouse environment.

 Project Overview

This system helps warehouse administrators efficiently manage:

* Suppliers
* Construction materials/products
* Customers
* Sales transactions
* Inventory records

The application demonstrates the practical use of Java OOP concepts such as classes, objects, encapsulation, and file/data management.

 Features

 Supplier Management

* Add suppliers
* View supplier details
* Update supplier information
* Delete supplier records

 Material Management

* Add construction materials
* Update material information
* Track available stock
* View material records

### Customer Management

* Register customers
* View customer details
* Manage customer records

 Sales Management

* Record sales transactions
* Calculate total sale amounts
* Update inventory after sales
* Generate sales reports

 Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Collections Framework
* Date and Time API
* Console-Based User Interface
